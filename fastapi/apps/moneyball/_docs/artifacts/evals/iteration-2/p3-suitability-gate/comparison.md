# Benchmark Comparison — P3 (적합성 게이트)

- eval name: p3-suitability-gate
- winner: tie (결론 동일) / with-harness 우위 (방법의 체계성)
- verdict: with-harness는 6차원 게이트를 표로 돌려 "5/6 손해 → 두꺼운 하네스 비추천 + 단일흐름 대안 + 게이트가 뒤집히는 조건"까지 체계적으로 판정. **그러나 baseline도 "함정이 있다 → 풀 하네스는 over-engineering, 안 만드는 안 추천"에 독립적으로 도달.** 즉 강한 모델은 직관으로 같은 결론에 옴. 게이트의 가치는 *결론*이 아니라 **매번 보장되는 체계성·명시성·뒤집힘 조건**.

## 한계 (정직)
테스트 프롬프트가 게이트 차원을 거의 그대로 불러줘서(딱 한 번/매번 다름/즉흥/주관적) baseline이 함정을 쉽게 알아챘다. **더 미묘한 near-miss(겉보기엔 반복 같지만 실제 검증 불가한 업무)로 다시 테스트해야** 게이트의 진짜 차별력이 보인다.
